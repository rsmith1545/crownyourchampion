🎵 Preview Link Generator

Run this once to bake the song previews into your site so every device (including iPhones) can play them.

  1. Click Generate Links below and let it run (about 2–3 minutes — it paces itself so Apple doesn't block it).
  2. When it finishes, click Download preview-urls.json.
  3. Upload preview-urls.json to your GitHub repo, in the same folder as index.html.
  4. Reload your site. Done — previews now play everywhere with no live lookups.

Tip: if it finds only a few, your network may be temporarily rate-limited from earlier testing — wait ~15 minutes (or try a different WiFi) and run it again.
