🎵 Preview Link Generator — Bruce Springsteen

Run this once to bake the song previews into your Springsteen site so every device (including iPhones) can play them.

  1. Click Generate Links below and let it run (about 2–3 minutes — it loads a few album catalogs first, then matches each song, pacing itself so Apple doesn't block it).
  2. When it finishes, click Download preview-urls.json.
  3. Upload preview-urls.json to your /springsteen/ folder, alongside its index.html.
  4. Reload the site. Done — previews now play everywhere with no live lookups.

It pulls previews from album tracklists and per-song search, with strict title-matching so it never grabs a live/acoustic take by mistake. Jersey Girl is pinned to Bruce's Live/1975–85 version (his only recording). If a song still can't be found, it just means Apple has no clean preview — the song still works in the bracket, just without a ▶.
